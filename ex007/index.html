Principais Formatações

Negrito/Destaque

Se eu quiser colocar em negito posso fazê-lo de duas formas:

Primeira forma, com a tag <b/>(sem semântica) em negrito

Segunda forma, com a tag <strong/>(com semântica) que deixa em negrito

Itálico/Ênfase

Nesta parte podemos usar a tag <i/>(sem semântica) o itálico para deixar

Ou podemos usar a tag <em/>(com semântica) para deixar em itálico

Texto Marcado

Podemos marcar um texto usando a tag <mark/>

Texto Grande e Pequeno

Podemos usar a tag <small/> para deixar o texto pequeno

Podemos usar a tag <big/> para deixar o texto GRANDÃO

Texto Deletado

Podemos deleter um texto com a tag <del/> assim

Texto Inserido

Podemos inserir um texto com a tag <ins/>assim

Texto Sublinhado

Podemos inserir a tag <u/>(não semântica) para deixar tudo sublinhado

Texto Sobrescrito

Podemos usar a tag <sup/> para colocar equações matemáticas, como por exemplo x20+10=y

Texto Subscrito

Podemos usar a tag <sub/> para deixar as palavras subscritas como por exemplo h2o
